g11n4j

Lightweight internationalization library for Java and Spring Boot.

Locale fallback, ICU4J plural rules, named/indexed placeholders, context-aware resolution, and runtime reload.

Modules

  • g11n4j-core - core Java library
  • g11n4j-spring-boot-starter - Spring Boot 3.x starter
  • g11n4j-spring-boot4-starter - Spring Boot 4.x starter

Supported formats

  • YAML (.yml, .yaml)
  • PROPERTIES (.properties)
  • JSON (.json)
  • GETTEXT (.po)
  • XLIFF (.xlf, .xliff)

Quick start

Add one of these dependencies in your project:

  • info.md7.g11n4j:g11n4j-core:1.0.0
  • info.md7.g11n4j:g11n4j-spring-boot-starter:1.0.0
  • info.md7.g11n4j:g11n4j-spring-boot4-starter:1.0.0

See repository README for full configuration and usage details.
